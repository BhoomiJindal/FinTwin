Onboarding & Privacy: Provide explicit, granular consent for the system to access financial data.  
Financial Aggregation: Link multiple bank accounts via an Account Aggregator to create a unified view of financial assets.  
Manual Asset Management: Add non-liquid assets like property, gold, and vehicles to compute a total net worth. 
Goal Setting: Define specific life goals (e.g., retirement, education, home purchase) with target amounts and timelines.  
Spending Analysis: View automated, categorized reports on income, spending habits, and saving trends.  
Personalized Financial Nudges: Receive real-time, actionable alerts regarding overspending, cash shortages, or opportunities to optimize savings.  
Investment Guidance: Access curated recommendations for SIPs, deposits, and tax-saving products based on a stated risk appetite.  
Portfolio Rebalancing: Receive strategic advice on shifting assets (e.g., gold to FD) based on real-time global economic indicators.  
Future Simulation: Interact with an AI-driven simulator to project wealth growth and model the impact of different saving/investing scenarios.  
Critical Action Execution: Initiate high-value financial actions such as modifying SIPs, moving surplus funds, or rebalancing portfolios.  
Real-time Security Transparency: Receive notifications when a "Wealth Protection Check" is actively evaluating a critical transaction.  
Risk-Based Feedback: View clear, non-technical warnings if an action is flagged as risky (e.g., "Action taken unusually fast").  
Action Mitigation: Acknowledge security warnings, follow system-mandated cooling-off periods, or provide secondary verification to proceed with an action.  
Explainable Blocking: View specific, transparent reasons if a transaction is automatically blocked by the protection layer, ensuring the user understands the "why" behind the decision.  
Audit Trail Review: Access a detailed log of all wealth-related actions, including the status (Allowed/Warned/Blocked) of each security-checked transaction.  
Logic Explainability: Click into specific financial recommendations to view the logic or rules used by the AI to generate that suggestion.
